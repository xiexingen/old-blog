Docker下安装redis

  1. 创建网络
    docker network create net_redis
  2. 创建redis配置文件(/docker/redis/config/redis.conf) 
    #redis的databases数量
databases 32
#redis密码
requirepass 123
  3. 通过docker-compose安装
    docker-compose -f redis-docker-compose.yml up -d
